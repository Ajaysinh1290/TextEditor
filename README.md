# Text Editor in java swing

### Hi there ...!
In this project I've created Text-Editor using Java-swing.

# Features
 
* Create file
* Save file
* Open file
* Change text color
* Change selection color
* Print file
* Change font style and size
* Cut-Copy-Paste
* Undo-Redo
* Find word
* Replace word
